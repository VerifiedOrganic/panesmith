# panesmith-vt100

`panesmith-vt100` implements the reference `vt100`-backed surface backend for
Panesmith panes.

## What this crate covers

Use this crate when you want deterministic ANSI parsing, replay support, and a
simple reference backend for tests, fixtures, and embedded previews.

## Feature flags

This crate does not define crate-specific feature flags today.
